Condicional Simples

Estruturas condicionais servem para para alterar o fluxo de execuçao do meu algoritimo. Após um teste de condicao, se a comdiçao for veradadeira, o fluxo de execuçao do codigo interno á estruturak condicional executa. Caso contrário, o codigo solta para o final da estrutura condicional

IF(TESTE DE CONDIÇAO){
Comandos executados quando a condicao é veradadeira
}else{ Comandos executados quando a condicao é falsa }
